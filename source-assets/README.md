# Local Reference Assets

This folder is intended for **locally extracted** Stardew Valley reference files when you want to regenerate some game-facing outputs.

These files are useful for development, but they do not need to be committed as part of the public repository by default.

For the current scripts, the most important optional local references are:

- `SmallFont.ru-RU.json`
- `SpriteFont1.ru-RU.json`

Depending on your workflow, you may also want local copies of:

- `Russian.fnt`
- `Russian_0.png`
- `font_bold.png`
- `font_colored.png`

The public source package focuses on the reconstructed glyph sets, the TTF family, build scripts, and documentation rather than on redistributing extracted game files.
